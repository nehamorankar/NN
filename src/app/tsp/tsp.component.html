Example:




Matrix form of Diagram :

-- A B C D
A -- 10 15 20
B 10 -- 35 25
C 15 35 -- 30
D 20 25 30 --


Solution:

Start from A:   [ A->B->D->C ]   Total disatnce = 80

Start from B:   [ B->A->C->D ]   Total disatnce = 80

Start from C:   [ C->A->B->D]   Total disatnce = 80

Start from D:   [ D->A->B->C ]   Total disatnce = 95

In this example Shortest path Total distance is 80 i.e we can start from A,B or C we get same disatnce.
